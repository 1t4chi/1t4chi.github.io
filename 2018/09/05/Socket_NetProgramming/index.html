网络程序设计模型

客户端/服务器模型 C/S

  • 非对称模型
  • 重复型服务器
  • 并发型服务器

浏览器/服务器模型 B/S

  • www 技术兴起，一种改进的 CS 结构，用于 web 应用
  • 主要采用 http https 协议进行数据传输
  • 三层结构
    • 客户端：即浏览器，负责客户和后台交互及其最终查询结果的输出
    • 服务器：功能层，与后台数据库连接 将由数据库返回的结果返回客户端
    • 数据库服务器：数据层，应客户请求进行各种查询和处理

客户端服务器通讯模式

面向连接

  • connection-oriented
  • 传输数据前建立稳定可靠数据通道
  • 可靠稳定，但占用资源多
  • TCP

无连接

  • connectionless
  • 不用建立通道
  • 不可靠，性能高
  • UDP
