PyQt5 + Pycharm + QtDesigner 搭建完美 Qt 开发环境

Posted on 2018-11-18 | In PyQt
对于独立开发或者平时的小练手来说，PyQt 简直是最最好用的界面开发工具了。本文记录了搭建 PyQt 开发环境和配置的全过程。

简介

PyQt是一个创建GUI应用程序的工具包。它是Python编程语言和Qt库的成功融合。它是一个多平台的工具包，可以运行在所有主要操作系统上，包括 Linux，Windows 和 Mac。比 wxpython 性能好一些

安装 PyQt5

Python3 支持 PyQt5;Python2 支持 PyQt4。
下载安装 PyQt 用 pip 是最简单的。

安装 Qt Designer

UI控件相关，可视化操作，直接拖拽控件，下载安装即可
Qt: http://iso.mirrors.ustc.edu.cn/qtproject/archive/qt/5.10/5.10.1/qt-opensource-mac-x64-5.10.1.dmg

Pycharm 配置

让 Pycharm 支持 QtDesigner 和 PyUIC

下面进行 PyQt 和 Pycharm 的配置。其中 PyUIC 是将 ui 文件转化为 Py 文件的工具，QtDesigner 搭建的文件是一个 ui 格式文件，需要转换。
进入 pycharm 的 Extern tools 栏目 setting->tools->external tools。
配置完后，在 pycharm 上方的 Tools 里面就可以看到 Qt Designer 这个扩展功能了，可以快速启动 Designer。

创建 UI 文件和将 UI 文件转为 Py 文件

  1. 创建新的UI文件：选中随便一个文件右键 Extern tools 选中 QtDesigner。
  2. 打开QtDesigner提示无法打开（当然了），直接新建，保存在当前工作目录。
  3. 打开某 UI 文件：选中它右键 Extern tools 选中 QtDesigner
  4. 编译 UI 为 py：右键 Extern tools 选中 PyUIC
