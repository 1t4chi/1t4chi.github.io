Hexo + NexT 美化实录

Posted on 2019-03-06 | In Blog
本文侧重Blog 的美化，包括更换背景、配色、图标、自定义界面、动画效果、添加分类与标签等等。

配置文件

Blog 目录下面有多个 _config.yml
一个在根目录 /_config.yml
其他的在主题目录，比如 next 的就在 /theme/next/_config.yml里

想要标签与分类界面

  1. 去掉 menu 注释
    在 next 主题目录下寻找 _config.yml ,找到 menu 栏去掉注释。
  • 位置：/theme/next/_config.yml

  1. 创建页面
    hexo new page 命令进行页面创建,主题的 _config.yml 文件中的 menu 中进行匹配，执行完后再/source 中出现 categories 和 tags 两个新目录，里面分别有一个 index.md。

  2. 添加 type 关键字
    上一步之后你会发现 categories 和 tags 页面出来了，但是没东西，侧边栏也没有链接。于是乎要在新建的 categories 和 tags 目录下的 index.md 中添加 type 关键字。

  • 配置位置：/source/categories/index.md 和 /source/tags/index.md

主题风格

配置位置：/theme/next/_config.yml 的 schemes 字段
默认是 Muse，这里我选了 Mist，去掉注释即可

如何设置头像

配置位置：/theme/next/_config.yml 的 Sidebar Avatar 字段
头像位置：头像命名为 header.jpg 放入/themes/next/source/images

头像太小了怎么破

配置位置：/themes/next/source/css/_variables/base.styl:
第一次设置完头像我一看只有 96px 方，这也太小了，我还要变成圆的呢，不更小了？
其实头像大小可以在上述 base.styl 中自定义
找到如下字段 site-author-image-width 修改后面的 px 值。下面的120px 是我改过的，感觉顺眼多了。

想要圆形头像？

配置位置：/themes/next/source/css/_common/components/sidebar/sidebar-author.styl
在 site-author-image 添加如下代码 border-radius: 60%;

还想要鼠标悬停旋转的那种？

配置位置：/themes/next/source/css/_common/components/sidebar/sidebar-author.styl

背景图片

插入背景图片

配置位置：themes\next\source\css\ _custom\custom.styl
这个是个性化定制样式的文件
推荐背景图片库连接如下
https://source.unsplash.com/
代码如下，插入 custom.styl 即可插入背景图片
background:url 可以换为任何你喜欢图片的地址，这里是 unsplash 的随机图片
repeat：是否重复出现
attachment：定义背景图片随滚动轴的移动方式
position：设置背景图像的起始位置

背景图片插入后有可能内容部分看不清？

可以设置 inner 部分解决，让你的博文内容部分有一层内背景，还可以设置透明度
